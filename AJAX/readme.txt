Although JSON is mostly used instead of XML
AJAX is not a language or a framework, it is a set of web technologies used to transfer data
XmlHttpRequest is 
  an API in the form of an object
  provided by browser's JS environment
  methods transfer data between clients and server
  can be used with other protocol's than HTTP
  can work with data other than XML(JSON, plain text)
